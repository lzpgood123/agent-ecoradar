## 产品愿景（固定置顶）

> 成为 AI Coding Agent 生态的权威索引——让每个开发者/Agent 都能快速找到最适合自己工具链的扩展资源。
> 核心定位：不是普通 awesome list，而是全自动采集、评分、推荐的生态追踪索引库。通过 GitHub 和正式站点双渠道交付，完全自动化运营。

## 用户偏好（固定置顶）

| 维度 | 偏好 | 说明 |
|------|------|------|
| 交互 | 自动化优先 | 全自动管道，无需人工逐条审核 |
| 视觉 | 简约 GitHub 风格 | 白底灰边框，功能优先 |
| 技术 | Python 纯脚本 | 无框架，无 LLM 调用，确定性规则 |
| 产品哲学 | 自动化 > 完美 | 自动化采集 80% 好过人工审核 100% |

## 标签说明

| 标签 | 含义 | 使用场景 |

| 💡想法 | 提出但未决定 | 随口提的创意、灵感、可能的改进方向 |
