# Wiki — Search in Coding 项目全息手册

> 如果只有 5 分钟，读 L1-全景 + L3-代码地图。
> 如果想完整理解项目，按层级从 L1 读到 L6。

## 一句话描述

Search in Coding 是一个面向终端型 / Agentic AI Coding 工具的持续自动更新生态追踪索引库——每天自动发现、归一化、评分、分类 AI Coding Agent 生态资源（工具本体、插件、MCP/ACP/A2A、skills、rules、prompts、上下文工程、PR review、CI 自动化、教程、案例和评测），通过 GitHub 总仓库和正式站点双渠道交付。

## 阅读路线图

**你是新对话，完全不了解项目？**
L1-全景 → L2-架构 → L3-代码地图 → 然后按需深入

**你是来修站点（前端）bug 的？**
L1-全景 → L3-代码地图 → L4A-前端详解 → L6-经验录

**你是来改采集/评分管道的？**
L1-全景 → L3-代码地图 → L4B-后端详解 → L5-接口契约 → L6-经验录

**你是来做架构决策的？**
L1-全景 → L2-架构 → L5-接口契约 → L6-经验录

**你是来写测试的？**
L3-代码地图 → L4B-后端详解（测试覆盖章节）→ L5-接口契约 → L6-经验录

**你是方案设计 agent，来讨论产品方向的？**
P1-产品决策日志 → L1-全景 → P3-框架演进 → 然后按需深入
