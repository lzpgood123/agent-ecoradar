# Prompt: Deep Dive Template

对一个 AI Coding 工具、项目、插件、MCP server、rules 仓库、skills 包或教程做深度分析。

## 输入

目标：`<填入名称或 URL>`

## 必须调查

1. 官方来源：官网、docs、GitHub、release、changelog。
2. GitHub 数据：stars、forks、license、last pushed、issues、topics、README。
3. 互联网资料：优先 Exa，找教程、评测、案例、争议。
4. 和目标工具的关系：Claude Code / Codex / Antigravity / OpenCode / Goose / Qoder / Trae / WorkBuddy / Cursor / Hermes。
5. 属于哪些概念：MCP、skills、rules、context engineering、multi-agent、CI、PR review、sandbox 等。
