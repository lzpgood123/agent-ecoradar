# Taxonomy

Top-level categories:

- `official-tool` — official product repos/docs/releases.
- `terminal-agent` — terminal-native coding agents.
- `ai-ide` — IDE/editor products with agent capabilities.
- `agent-harness` — orchestration/runtime/harness systems.
- `rules-instructions` — CLAUDE.md, AGENTS.md, Cursor rules, OpenCode rules.
- `skills-prompts` — skills, slash commands, prompt packs, reusable workflows.
- `mcp-acp-a2a` — MCP/ACP/A2A servers, adapters and registries.
- `context-engineering` — repo maps, code graph, indexing, RAG, memory.
- `testing-review-ci` — PR review, code review, CI agents, test generation.
- `multi-agent-workflow` — planner/builder/reviewer teams, subagents.
- `tutorial-case-study` — tutorials, blog posts, videos, experience reports.
- `benchmark-evaluation` — benchmarks, comparisons and empirical evaluations.
- `ui-dashboard` — visual control panels and session managers.
- `security-sandbox` — permissions, sandboxing, prompt-injection defenses.
