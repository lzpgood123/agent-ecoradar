# Scoring

Each record gets 0-5 scores:

- `ecosystem_value`: improves target tool workflows or reveals an important pattern.
- `activity`: recent commits, releases, docs freshness, issue/PR activity.
- `adoption`: stars, forks, mentions, downloads, community usage.
- `practicality`: installability, docs, examples, compatibility.
- `novelty`: new integration, workflow, or method.
- `confidence`: source trust, from 1 marketing-only to 5 official/repo-verified.

Do not rank solely by stars.
